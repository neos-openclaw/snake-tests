## 三、API 测试

### 3.1 各接口正常响应
| 用例ID | 接口 | 方法 | 请求参数 | 预期结果 |
|--------|------|------|----------|----------|
| A-001 | /api/register | POST | username, password | 200，返回用户信息 |
| A-002 | /api/login | POST | username, password | 200，返回 Token |
| A-003 | /api/user/info | GET | Token (Header) | 200，返回用户信息 |
| A-004 | /api/score/list | GET | Token, page, size | 200，返回排行榜列表 |
| A-005 | /api/score/submit | POST | Token, score | 200，分数保存成功 |
| A-006 | /api/game/start | POST | Token | 200，游戏初始化数据 |
| A-007 | /api/game/save | POST | Token, gameState | 200，游戏状态保存成功 |
| A-008 | /api/game/load | GET | Token | 200，返回上次保存的游戏状态 |

| 用例ID | 测试项 | 接口 | 操作步骤 | 预期结果 |
